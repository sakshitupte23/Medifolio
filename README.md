# MediFolio – Healthcare Management System

A modern **Healthcare Management System** that streamlines patient record management, appointment scheduling, biometric authentication, and secure medical data storage. MediFolio is designed to improve healthcare workflows by providing a centralized platform for patients, doctors, and administrators.

---

## 📌 Overview

MediFolio is a web-based healthcare application that enables efficient management of medical records while ensuring security through biometric fingerprint authentication. The system integrates Firebase for real-time cloud data storage and Raspberry Pi for biometric verification.

## 🚀 Getting Started

### Prerequisites

* Modern Web Browser
* Firebase Account
* Visual Studio Code (Recommended)
* Git
* Raspberry Pi (Optional for biometric authentication)

### Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/MediFolio.git
```

2. Navigate to the project folder

```bash
cd MediFolio
```

3. Configure Firebase

* Create a Firebase project.
* Enable Authentication.
* Enable Realtime Database.
* Replace the Firebase configuration in:

```javascript
firebase-config.js
```

4. Run the project

Open `index.html` using Live Server or your preferred local server.
